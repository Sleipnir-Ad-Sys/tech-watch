Comment ça fonctionne

01

Collecte automatique

Le pipeline Python récupère chaque matin les nouvelles releases GitHub, les articles RSS et les packages PyPI des projets configurés.

→
02

Analyse & scoring

Polars + DuckDB calculent un score d'impact basé sur les stars, la fréquence de release et l'activité de la communauté.

→
03

Déploiement GitHub Pages

Les données sont exportées en JSON statique et publiées automatiquement. Aucun serveur nécessaire.
